# Gemini (CLI, Code Assist, Antigravity, etc.)

| File | Purpose |
|---|---|
| `GEMINI.md` | Project-level instructions |
| `.opstwin/rules.md` | Full audit schema |
| `.opstwin/task-template.md` | Per-task prompt |

## Usage

1. Add `.opstwin/` and `GEMINI.md` to your project
2. Include `.opstwin/rules.md` in your system or project context
3. Fill in `.opstwin/task-template.md` and paste as your task
4. Set `"agent": "gemini"` in the audit JSON
