# Gravity

| File | Purpose |
|---|---|
| `GRAVITY.md` | Gravity project instructions |
| `.opstwin/rules.md` | Full audit schema |

## Usage

1. Init OpsTwin in your repo
2. Point Gravity at `GRAVITY.md` or `AGENTS.md` as project context
3. Fill in `.opstwin/task-template.md` for each task
4. Set `"agent": "gravity"` in the audit JSON
